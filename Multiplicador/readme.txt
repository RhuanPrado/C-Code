Esse código em C é um multiplicador de números inteiros maiores que podem ser armazenado em memória
Para números inteiros, que chamamos de tipo int, juntamos 4 bytes (32 bits)
podemos ter números de 0 a 4,294,967,295 e chamamos de tipo unsigned int.

para realizar a multiplicação de numeros maiores que podemos utilizar em memória nesse código temos a demonstração da utilização 
de ponteiros para realização desse cálculo imprimindo em um arquivo de saída.
